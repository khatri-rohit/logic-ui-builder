# components/ui/ - shadcn/ui Components

Radix UI + Tailwind v4 component library.

## FILES

- Main components in `components/ui/`
- Managed via shadcn CLI
- `components.json` - shadcn config

## COMPONENTS

- Button, Dialog, DropdownMenu, Input, Select, etc.
- Using `class-variance-authority` for variants

## ANTI-PATTERNS

- DO NOT use `npx shadcn@latest` - use `npx shadcn`
- DO NOT add raw CSS - use Tailwind tokens
